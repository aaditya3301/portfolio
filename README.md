# Portfolio

Welcome to my Portfolio! This is a website that showcases my skills, experience, and projects. Built with HTML, CSS, JavaScript, and Tailwind CSS, it is designed to be fully responsive and optimized for various screen sizes.

## Sections

- **Home:** An introductory section with a profile picture and a brief statement about myself.
- **About:** Details about my skills, technologies I work with, and a short description of my journey as a frontend developer.
- **Experience:** A section that highlights my experience as a developer, including projects and key achievements.
- **Projects:** A gallery of my projects with links to view them on GitHub and live demos.
- **Get in Touch:** A contact form for visitors to reach out to me.

## Technologies Used

- HTML
- CSS
- JavaScript
- Tailwind CSS
- Bootstrap 

## Features

- **Responsive Design:** Optimized for mobile and desktop devices.
- **Smooth Animations:** Subtle animations for better user interaction.
- **Clean UI:** A modern, minimalist design with a focus on clarity.
- **Interactive Sections:** Sections like "Projects" include clickable links to view code on GitHub and demo live websites.
