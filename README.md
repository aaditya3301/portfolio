# Portfolio - Aaditya Singhal

 Portfolio website showcasing AI & Cloud Engineering projects with premium dark theme and interactive animations.

## Features

- **Custom Cursor** - Magnetic cursor with smooth follower effect
- **Smooth Scroll** - Buttery smooth scrolling powered by Lenis
- **3D Tilt Cards** - Interactive project cards with depth
- **Text Scramble** - Dynamic hero text animation
- **GSAP Animations** - Professional gradient orb animations
- **Scroll Progress** - Visual reading progress indicator
- **Accessibility** - Respects `prefers-reduced-motion`

## Tech Stack

- **Framework:** Next.js 16 (Turbopack)
- **Animations:** Framer Motion, GSAP, Vanilla Tilt, Lenis
- **Styling:** Custom CSS with modern design system
- **Deployment:** Vercel
